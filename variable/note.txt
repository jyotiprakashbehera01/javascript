  1.PRIMITIVE :

  ->Definition:
  ->Primitive data types are the basic built-in data types in JavaScript.
  ->They store a single value and are immutable,
    which means their values cannot be changed after they are created.
  -> If you modify a primitive value, JavaScript creates a new value instead of changing the original one.

  2.NON_PREMETIVE:

  Definition:
  ->Non-primitive data types are data types that can store multiple values or collections of data.
  ->They are mutable, which means their contents can be changed after they are created.
  ->Non-primitive values are stored by reference, not by value.
  ->The main non-primitive data type in JavaScript is Object.
